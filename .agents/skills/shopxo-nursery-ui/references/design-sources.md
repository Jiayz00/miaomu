# Design guidance sources

This project Skill was authored for the ShopXO nursery requirements after reviewing these open-source sources:

- Anthropic `frontend-design` Skill: `https://github.com/anthropics/skills/tree/main/skills/frontend-design`, Apache License 2.0. Used as process inspiration for subject-specific direction, critique before implementation, restraint, accessibility, and screenshot review; the project text and nursery rules are independently adapted.
- Local `redesign-existing-projects` Skill: used as an audit checklist for typography, layout, interaction states, responsive behavior, and preservation of the existing stack.

When updating this Skill, review upstream changes manually. Do not execute or copy community scripts without a source and license audit, and never install them into the user-level Codex directory for this project.
